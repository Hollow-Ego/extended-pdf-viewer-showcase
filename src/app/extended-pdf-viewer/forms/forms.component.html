Configuration:
If you're using the default options, you don't have to do anything to activate the feature. If you're using custom options, make sure to activate the setting renderInteractiveForms: true.

`[(formData)]` links the fields in the PDF form with your Angular component. Note that `(formDataChange)` reports every filled input field of the PDF file, including fields you didn't send with `[formData]`.
